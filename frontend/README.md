# RIDA - Raindrop Intelligent Document Agent

RIDA is a next-generation document intelligence console built for the **AI Championship Hackathon**.

## Features
- **Smart Ingestion**: Automatically classifies and processes invoices, contracts, and more.
- **Semantic Memory**: Chat with your entire document repository.
- **Vultr Integration**: Secure, high-performance storage for your critical documents.
- **Raindrop Powered**: Built on the Raindrop Platform using SmartBuckets, SmartInference, and SmartSQL.

## Tech Stack
- **Frontend**: Vite + React, Tailwind CSS, shadcn/ui
- **Backend**: Express.js (Raindrop MCP)
- **Infrastructure**: Vultr Object Storage

## Getting Started
1. `npm install`
2. `npm run dev`
